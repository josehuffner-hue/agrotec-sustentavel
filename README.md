# Projeto Agrinho 2026 — Escola José Pacheco

Este repositório hospeda o site oficial do **Projeto Agrinho 2026**, desenvolvido com foco em sustentabilidade, inovação rural e educação comunitária, inspirado nos ideais pedagógicos de **José Pacheco**.

## 🛠️ Tecnologias Utilizadas
- **HTML5** (Estruturação de conteúdo)
- **CSS3** (Design responsivo e identidade visual ecológica)
- **JavaScript** (Interatividade e dinamismo)

## 📦 Como Visualizar o Projeto
1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` em qualquer navegador.
3. Certifique-se de que todos os arquivos estão na mesma pasta.
