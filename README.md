# 🌱 Agro Forte, Futuro Sustentável

## Sobre o Projeto

Este site foi desenvolvido para o Projeto Agrinho 2026 com o tema **"Agro Forte, Futuro Sustentável"**.

O objetivo é apresentar a importância do agronegócio para a sociedade, destacando práticas sustentáveis que ajudam a preservar o meio ambiente e garantem um futuro melhor para as próximas gerações.

## Objetivos

* Mostrar a importância do agro para a economia e alimentação.
* Incentivar a preservação dos recursos naturais.
* Promover a conscientização ambiental.
* Valorizar a relação entre campo e cidade.
* Destacar a sustentabilidade como caminho para o futuro.

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* GitHub Pages

## Funcionalidades

* Página informativa sobre o Agrinho 2026.
* Design responsivo para computadores e celulares.
* Navegação simples e intuitiva.
* Conteúdo educativo sobre sustentabilidade.
